## Troubleshooting

### Map doesn't appear?
1. Check that you've selected a region
2. Click "View Regional Map" button
3. If still not showing, refresh the page

### Communities not showing?
1. Zoom out (scroll down)
2. Pan around (drag the map)
3. Make sure you can see the entire map area

### Interactions not working?
1. Try refreshing the page
2. Clear browser cache
3. Try a different browser

### Colors look wrong?
1. Make sure you're on the right metric
2. Check the color scale (green=good, red=needs work)
3. Refresh if colors seem stuck

### Performance slow?
1. Close other browser tabs
2. Disable browser extensions
3. Refresh the page
